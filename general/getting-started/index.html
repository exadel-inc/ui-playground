    Project structure

    UIP components are organized in the following way:

    • Base components
    • Plugins
      • UIPEditor
      • UIPOptions
      • UIPSettings
        • UIPBoolSetting
        • UIPSelectSetting
        • UIPTextSetting
      • UIPHeader
        • UIPOptions
        • UIPSnippets

    Any playground must have at least core components. Plugins are optional, you can add them on your own free will.

    To implement custom UIPPlayground components, see UIPPlugin.
